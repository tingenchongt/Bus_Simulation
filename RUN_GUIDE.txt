THESIS SIMULATION — RUN GUIDE (UXsim, aligned with current code)
==================================================================

QUICK RUN
---------
  cd to this folder (simulation project root), then:
    pip install uxsim openpyxl matplotlib
    python simulation.py

  That executes 12 UXsim scenarios (3 Excel sheets x 4 runs each), writes CSVs,
  copies network PNGs, and builds bar charts in figures_output/analysis/ when matplotlib works.

  Charts only (CSVs already exist):
    python visualize_results.py

  Export Chapter 4 to Word (.docx) from THESIS_DRAFT.md:
    python export_chapter4_docx.py
    Output: THESIS_Chapter4.docx (or THESIS_Chapter4_generated.docx if Word has the file open)

  Excel calibration printout only (no simulation):
    python bus_calibration.py


WHAT THE MODEL IS (for your paper)
----------------------------------
  Corridor: Robinsons <-> Waltermart on Emilio Aguinaldo Highway (schematic).

  Stops / links:
    - Robinsons terminal bay (dwell from Excel mean at Robinsons for that sheet).
    - Baseline only: informal CURB on Robinsons->Waltermart BEFORE the signal
      (dwell proxy; not a formal bay).
    - Signal node: 90 s cycle, 45 s EB green, 45 s WB green.
    - Formal bus stop AFTER the intersection: 290 m cruise from signal to stop (tune on Maps).
    - Waltermart terminal bay (dwell from Excel mean at Waltermart for that sheet).
    - Waltermart->Robinsons uses post-intersection stop + signal; no informal leg.

  Mainline cruise budget Robinsons_road <-> Waltermart_road: 3.3 km (see TOTAL_CRUISE_RB_ROAD_TO_WM_ROAD_M).
  Cruise legs before/after informal and signal are tunable in simulation.py.

  Twelve runs per full execution:
    For each sheet (Morning Session, Lunch, Afternoon Session):
      1) robinsons_location + baseline (informal on)
      2) robinsons_location + optimized (informal off, dwell scale 1.0)
      3) waltermart_location + baseline (informal on; window from WM rows)
      4) waltermart_location + optimized (informal off; dwell scale 0.55)

  Volumes: RB->WM trips = count of rows with Location at Robinsons on that sheet;
           WM->RB trips = count of rows with Location at Waltermart on that sheet.


OUTPUT FILES (paste into thesis / Excel)
----------------------------------------
  results_summary.csv
    One row per run (12 rows): session, data_origin, policy, informal flag,
    short_dwell_scale, volumes, subset survey summaries, travel time, delay, etc.

  results_baseline_vs_optimized.csv
    Six rows: per session AND per data_origin (robinsons vs waltermart),
    baseline vs optimized travel time and percent improvement.

  figures_output\
    One network_average PNG per scenario (long filenames include session and policy).

  figures_output/analysis\
    travel_time_robinsons.png, travel_time_waltermart.png,
    pct_travel_time_improvement.png, delay_robinsons.png, travel_time_all_runs.png

CODE YOU MAY TUNE (simulation.py)
---------------------------------
  TOTAL_CRUISE_RB_ROAD_TO_WM_ROAD_M — 3300 m interior (verify on Google Earth).
  DIST_INTERSECTION_TO_POSTINT_CRUISE_M — 290 m signal to post-intersection stop.
  CRUISE_RB_TO_INFORMAL_M, CRUISE_INFORMAL_TO_SIGNAL_M — chainage before/after informal curb.
  SIGNAL_GREEN_EB_S, SIGNAL_GREEN_WB_S — greens (sum = cycle length).
  OPTIMIZED_SHORT_DWELL_SCALE — 0.55 on Waltermart-stream optimized runs only.
  SHOW_NETWORK_WINDOW — True pops a window each run (usually leave False).


CODE YOU MAY TUNE (bus_calibration.py)
--------------------------------------
  MAX_DEMAND_WINDOW_CAP_S — caps demand window when Excel times span too many hours (default 5 h).
  Set to None to disable capping.

EXTENDED STUDY (3049 m corridor, mixed traffic, signal scenarios) — NEW
-----------------------------------------------------------------------
  Field chainage + GPS are in corridor_config.py (from your route notes):
    0 m Robinsons | 50 m informal Robinsons | 200 m 7-Eleven | 1845 m Villa Verde |
    2083 m Vista Mall | 2495 m signal (45/45) | 2884 m RCBC | 3049 m Waltermart.

  Formal stops (survey legend): Robinsons, Waltermart, Vista Mall.
  Informal stops: 7-Eleven, Villa Verde, RCBC Bank.

  Run:
    python simulation_extended.py --quick          # 4 runs, bus only, 1 signal scenario
    python simulation_extended.py --all-signals      # 48 runs (3 sessions x 4 policies x 4 signals)
    python simulation_extended.py --no-mixed         # buses only (default loads all modes)

  Mixed traffic (default ON): bus + private car + jeepney + truck + motorcycle + van.
  Non-bus trips use mainline bypass links (no stop dwell). Tune volume_multiplier in
  corridor_config.py VEHICLE_CLASSES (Excel has no per-mode counts yet).

  Signal scenarios (4 labels, one physical signal at 2495 m):
    sig1_G_sig2_G | sig1_G_sig2_R | sig1_R_sig2_G | sig1_R_sig2_R
    Uses UXsim signal_offset (first encounter EB) + WM->RB demand time shift (second encounter WB).
    Thesis should state this is a controlled sensitivity, not measured phase logs.

  Outputs:
    results_summary_extended.csv
    results_baseline_vs_optimized_extended.csv
    figures_output/*_extended* or long scenario filenames

  Original simplified 3.3 km / 12-run thesis block is unchanged in simulation.py.
